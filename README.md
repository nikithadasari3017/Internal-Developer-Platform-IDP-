🚀 Internal Developer Platform (IDP)

Simplifying cloud deployment through automation.

An AWS-based platform designed to help developers deploy applications with minimal manual infrastructure configuration. The platform automates EC2 provisioning, server setup, application deployment, monitoring, and environment management through a centralized dashboard.

✨ Features
🔐 User authentication and role-based access
📦 Application registration and management
☁️ Automated AWS EC2 provisioning
⚡ One-click application deployment
🧩 Deployment templates for PHP, Laravel, Flask, Node.js, and WordPress
📜 Deployment logs and status tracking
📊 Server monitoring and application lifecycle management
🧹 Environment cleanup to reduce unnecessary AWS resource usage
🛠️ Tech Stack

Frontend: HTML, CSS, Bootstrap, JavaScript
Backend: Python, Flask
Database: MySQL
Cloud: AWS EC2, IAM, Security Groups
Automation: Boto3
Version Control: Git & GitHub

🔄 Deployment Workflow

Register Application → Provision AWS Infrastructure → Configure Server → Deploy Application → Monitor → Manage Environment

🎯 Project Goal

To reduce the complexity of manual cloud deployments and provide developers with a simple, centralized platform for managing the complete application deployment lifecycle.
